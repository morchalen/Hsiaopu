## 图 2：AI 对话与工具调用流程图

```mermaid
sequenceDiagram
    participant User as 用户
    participant UI as HomeScreen
    participant VM as ChatViewModel
    participant Registry as AiProviderRegistry
    participant Provider as DeepSeekProvider
    participant Bus as ShellCommandBus
    participant Shell as ShellScreen
    participant System as SystemControlExecutor

    User->>UI: 输入消息 "帮我查看电量"
    UI->>VM: sendMessage(content)
    
    VM->>VM: 构建系统提示词(含工具列表)
    VM->>Registry: sendMessageStream(messages)
    Registry->>Provider: sendMessageStream(messages)
    Provider-->>VM: 流式回复 "[TOOL:get_battery_info]"
    
    VM->>VM: 解析 [TOOL:get_battery_info]
    VM->>VM: getShellCommandForAction("get_battery_info")
    VM-->>VM: 返回命令 "dumpsys battery"
    
    VM->>Bus: sendCommand("dumpsys battery")
    Bus-->>Shell: commands.collect(cmd)
    
    Shell->>System: executeCommand("dumpsys battery")
    System-->>Shell: ShellResult(stdout, stderr)
    
    Shell->>Bus: sendResult(result)
    Bus-->>VM: results.first { command == cmd }
    
    VM->>VM: 将工具结果拼接到回复
    VM->>Registry: sendMessageStream(第二轮请求)
    Registry->>Provider: sendMessageStream(总结请求)
    Provider-->>VM: 流式回复 "电量已查询..."
    
    VM->>UI: 更新 messages + streamingContent
    UI-->>User: 显示 AI 回复
```
